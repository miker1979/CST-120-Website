CST-120 Web Development Syllabus

Course Description

This course introduces the fundamental concepts and syntax of the web development languages including HTML, CSS, and JavaScript. The course focuses on foundation required to build complex dynamic web applications. Prerequisite: none.

Weeks 1–2

Topic 1 HTML

Description: In this topic, students are introduced to the HTML markup language where they learn basic structure, basic elements of an HTML document, and how to work with text and basic controls. The students finish this topic by learning the developer tools available to them, as well as how web servers work to help support development of HTML-based applications.

Objectives

  1. Develop webpages with basic elements and structure.
  2. Develop webpages with text including headings, paragraphs, formatting, fonts, and colors.
  3. Develop webpages with components including links, images, and lists.
  4. Investigate how a web server functions.
  5. Investigate the developer tools available to a webpage developer.

Readings

  • W3Schools HTML Home Tutorial

View Course Assignment Matrix

Weeks 3–4

Topic 2 Advanced HTML

Description: In this topic, students continue to expand their knowledge of the HTML markup language, learning some of the more advanced features of the HTML markup language and how to work with layouts, graphics, and media content. The students finish this topic by learning how to make HTML pages ADA compliant.

Objectives

  1. Develop webpages with layouts using blocks, divs, spans, and tables.
  2. Develop webpages with graphics including image, canvas, and SVG tags.
  3. Develop webpages with media including video and audio tags.
  4. Explain how and why ADA compliance is achieved with ADA guidelines.

Readings

  • W3Schools HTML Forms Tutorial
  • W3Schools HTML Graphics Tutorial
  • W3Schools HTML Media Tutorial
  • W3Schools HTML APIs Tutorial

Weeks 5–6

Topic 3 CSS

In this topic, students learn Cascading Style Sheets (CSS) and how to apply styling to HTML pages using fonts, colors, and formatting. The students finish the topic by learning how to apply backgrounds, transitions, and animations to their HTML pages.

Objectives

  1. Develop webpages with CSS basic styles including inline, internal, and external styles.
  2. Develop webpages with text styles including fonts, colors, and alignment.
  3. Develop webpages with formatting and layout styles including margins, padding, and borders.
  4. Develop webpages with transitions and animations.

Readings

  • W3Schools CSS Tutorial
  • W3Schools CSS Advanced Tutorial

Weeks 7–8

Topic 4 JavaScript

In this topic, students are introduced to the JavaScript programming language. This language is used as the foundation for client-side programming on webpages. Students learn the basic language syntax, how to work with variables, make decisions, perform repetitive operations, and create reusable functions.

Objectives

  1. Develop code that integrates basic JavaScript in a webpage.
  2. Develop code that leverages variables and data types.
  3. Develop code that leverages operators.
  4. Develop code that uses controls such as if statements and loops.
  5. Develop code that uses reusable functions.

Readings

  • W3Schools JavaScript Tutorial

Weeks 9–10

Topic 5 DOM and Events

In this topic, students learn how JavaScript interacts with the webpage through the Document Object Model (DOM). Students practice selecting elements, changing content and styles, and responding to user actions through events.

Objectives

  1. Explain the purpose of the DOM.
  2. Develop code that selects and manipulates DOM elements.
  3. Develop code that responds to events such as clicks and input changes.
  4. Develop code that validates user input.

Readings

  • W3Schools JavaScript HTML DOM Tutorial
  • W3Schools JavaScript HTML DOM Events Tutorial

Weeks 11–12

Topic 6 Bootstrap

In this topic, students learn how to use the Bootstrap framework to create responsive webpages. Students practice using Bootstrap containers, the grid system, and common components to create modern page layouts.

Objectives

  1. Explain what Bootstrap is and why it is used.
  2. Develop webpages using Bootstrap containers and the grid system.
  3. Develop webpages using common Bootstrap components.
  4. Develop webpages that are responsive across different screen sizes.

Readings

  • W3Schools Bootstrap Tutorial

Weeks 13–14

Topic 7 Final Project

In this topic, students apply the skills learned throughout the course to build a final project. Students plan, develop, and refine a multi-page website that demonstrates HTML, CSS, and JavaScript concepts.

Objectives

  1. Plan and organize a multi-page website project.
  2. Develop a website that uses HTML, CSS, and JavaScript.
  3. Test and refine the website based on requirements.
  4. Present the final project.

Readings

  • Review prior W3Schools tutorials as needed.
